## 1. Maintenance Informatique - Projet L1

**Chemin :** `/Maintenance-informatique-projet-L1`

### Description
Projet de première année (L1) axé sur les bases de la maintenance informatique. Ce projet couvre les concepts fondamentaux de diagnostic, dépannage et maintenance des systèmes informatiques.

### Sujets couverts
- Diagnostic des pannes matérielles et logicielles
- Maintenance préventive et corrective
- Gestion des ressources système
- Documentation des interventions

### Prérequis
- Connaissance de base en informatique
- Outils de diagnostic système (BIOS, gestionnaire des tâches, etc.)

## 2. Linux Ubuntu Server

**Chemin :** `/linux-ubuntu-server`

### Description
Projet dédié à l'administration et la configuration d'un serveur Ubuntu Linux. Découvrez comment configurer, gérer et maintenir un serveur Linux en environnement de production.

### Sujets couverts
- Installation et configuration d'Ubuntu Server
- Gestion des utilisateurs et permissions
- Configuration réseau (IP statique, DNS, firewall)
- Gestion des services et daemons
- Sécurité et hardening du serveur
- Gestion des paquets et mises à jour

### Prérequis
- Accès à une machine virtuelle ou serveur Ubuntu
- Connaissance de base de Linux et du terminal
- Notions de réseau TCP/IP

## 3. Routage IP - Projet L2

**Chemin :** `/routage-ip-projet-l2`

### Description
Projet de deuxième année (L2) focused sur les concepts avancés du routage IP. Ce projet explore la configuration de routeurs, la gestion des tables de routage et les protocoles de routage.

### Sujets couverts
- Architecture et principes du routage IP
- Configuration statique des routes
- Protocoles de routage dynamique (OSPF, BGP, RIP)
- Tables de routage et métriques
- Simulation de topologies réseau complexes
- Optimisation des chemins réseau

### Prérequis
- Compréhension solide des concepts IP et TCP/IP
- Connaissance des modèles OSI et des couches réseau
- Accès à un simulateur réseau (Cisco Packet Tracer, GNS3, etc.)

### Outils recommandés
- Cisco Packet Tracer
- GNS3 (Graphical Network Simulator)
- Wireshark (pour l'analyse de trafic)

## 4. Windows Server Core 2022

**Chemin :** `/windows-server-core-2022`

### Description
Projet dédié à l'administration de Windows Server 2022 en mode Core. Apprenez à gérer, configurer et maintenir un serveur Windows Server dans un environnement professionnel.

### Sujets couverts
- Installation de Windows Server 2022 Core
- Configuration du serveur (nom, adresse IP, DNS)
- Gestion des rôles et services serveur
- Active Directory et gestion des utilisateurs
- Configuration DHCP et DNS
- Sécurité et pare-feu Windows
- Gestion des partages de fichiers
- Monitoring et maintenance du serveur

### Prérequis
- Accès à Windows Server 2022 (VM ou physique)
- Notions de base en administration Windows
- Compréhension des concepts de domaine Active Directory
